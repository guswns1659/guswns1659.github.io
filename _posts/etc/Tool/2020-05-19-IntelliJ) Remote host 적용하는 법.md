다양한 Tool은 개발자의 생산성을 높여줍니다.

## 개요 
인텔리 J 속 remote host 기능에 대해 정리한 글 

## 사용법 
- 인텔리J 상단 Tools -> Deployment -> Browse Remote host
- SFTP 선택 후 SSH configuration에서 ... 클릭 
- Host에 ec2 ip 입력 
- User name : ubuntu ( ec2 만들 때 OS) 
- Authentication type : Key pair로 설정 
- pem key 위치를 찾는다. 
- Root Path : 파일을 이동할 폴더 위치 지정
