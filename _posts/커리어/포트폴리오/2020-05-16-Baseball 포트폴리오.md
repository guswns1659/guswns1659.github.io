### 제작동기
- 복잡한 야구 게임 구현을 통해 데이터 베이스 설계 능력 향상. 
- 로그인 기능을 위한 기술, OAuth2에 대한 이해도 향상.
### 개발환경
#### 기술 스펙
- Spring Boot 2.2.6 
- JDK 1.8
- Mysql 5.8
- Gradle 6.3
- AWS ec2 
- AWS RDS 
- Nginx 1.14.0
- GitHub OAuth2

#### 선택 이유
- 내장 톰캣으로 편한 배포를 하기 위해 스프링부트 사용.
- 최근 각광받는 AWS 클라우드 플랫폼을 사용하기 위함. 
- 사용자 정보를 저장하는 부담을 줄이는 OAuth2 기술 사용하기 위함.
