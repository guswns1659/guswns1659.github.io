## 잘먹었습니다. API

### 도메인 객체
-Project, Corporation, Feed, Restaurant, Account, Image, Application, Information, PointHistory

### 도메인 객체 수정사항
- 절감된 탄소량은 비워진 접시 * 0.1kg로 계산하기. 따로 객체 필드로 추가하지 않기

### DB에 넣을 데이터
- Account 3개 데이터
- Challenge 10개 데이터 : 빈그릇 구해서 넣어놓기.
- Project 5개 데이터 :
- 주변 식당 데이터 :


### 메인
- 오늘 비워진 접시 수와 절감된 탄소량
- 이번주 빈그릇 7개 (좋아요 수 내림차순)
- 진행중인 프로젝트 5개 (createdAt 최신순)
- 내 주변 친환경 식당 찾기
    - 요청 바디 : 사용자의 경도, 위도
    - 응답 바디 : 사용자 위치 nkm 이내 식당 리스트
- 어플리케이션 이용방법
    - 이미지 배열 응답

### 지도 상세보기
- 내 주변 친환경 식당 찾기 3개
    - 요청 바디 : 사용자의 경도, 위도
    - 응답 바디 : 사용자 위치 nkm 이내 식당 리스트

### 기부하기
- 마감 얼마남지 않은 프로젝트 3개(deadLine이 적은 순)
- 도움이 필요한 프로젝트 (전체 프로젝트)
- 프로젝트 상세보기
- 기부하기 버튼 누르면 사용자의 포인트 보여주기
- Post 기부하면 프로젝트 금액 증가.
  - 요청 바디 : 프로젝트에 기부하는 에코 포인트, 해당 프로젝트 Id,
  - 응답 바디 : 에코 포인트 추가된 프로젝트 객체 응답

### 피드
- 전체 피드
- 피드 상세페이지
- Post 좋아요 버튼 누르면 좋아요 증가.
  - 요청 바디 : 좋아요 1
  - 응답 바디 : 좋아요 추가된 피드 객체 응답

### 마이페이지
- 오늘 참여 횟수와 총 참여횟수
- 포인트 히스토리 최신순

### 사진 업로드
- 사진 촬영 후 데이터 넘어오면 s3에 저장된 url 응답.

### 프로젝트 예시
- 북극곰 살리기 1,000만원
- 개마고원에 나무 심기 1,000만원
- 순천만 습지에 맹꽁이 살리기 1,000만원
- 해양쓰레기 줍기 프로젝트 1,000만원
- 해양 포유류 보호법 후원 프로젝트 1,000만원
