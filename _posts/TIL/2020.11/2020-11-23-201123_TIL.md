제일 무서운 사람은 매일 꾸준히 무언가를 하는 사람

# 맥북
- Command + ↓ :  파일, 폴더 들어가기
- enter : 이름 바꾸기
- cmd + [ : 뒤로가기

# Vim

- ; : f로 해당 단어를 찾은 후 같은 단어로 이동할 때 사용
- , : ;로 이동한 단어를 다시 앞으로 찾을 때 사용
- n : /로 단어를 찾은 후 해당 단어로 이동할 때 사용.
- /asdf : 단어 찾기 하이라이트 모드에서 나올 때!
- ctrl + e, y : 커서는 그대로 두고 스크롤할 때
- ctrl + u, d : 반페이지씩 이동
- :12 : 12행으로 이동
- ctrl + h : 입력모드에서 단어 지우는 단축키()
