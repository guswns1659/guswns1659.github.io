## 백준 알고리즘 제출 시 주의사항
- 리턴이 아니라 출력을 해야 하기 때문에 오답 조건일 때 출력만 한다고 끝나지 않는다. 최종적으로 오답을 출력해야 한다. boolean flag를 이용해 알고리즘이 끝나고 flag에 따라 정답, 오답을 출력한다.

```java
if (!ok) {
            System.out.println("NO");
        } else {
            answers.forEach(System.out::println);
        }
```
